Quick Guide: Running Windup on Source

  1. Download Windup and Unzip to your favorite directory.
  2. Edit the jboss-windup.sh, with the -input parameter set to your source directory.
  3. Make sure to set your -javaPkg parameter to your Java package structure.
  4. Make sure to set the -source true flag.
  5. Make sure to set your -output parameter to your favorite output directory for the report.
  6. Execute the jboss-windup.sh
  7. View the report!

Quick Guide: Running Windup on Archives

Before you get started: Make sure you setup your favorite decompiler to run in Archive mode! | See How!

  1. Download Windup and Unzip to your favorite directory.
  2. Edit the jboss-windup.sh, with the -input parameter set to a directory containing your archive.
  3. Make sure to set your -javaPkg parameter to your Java package structure.
  4. Execute the jboss-windup.sh
  5. View the report!
