Quick Guide: Running Windup

  1. Download Windup and Unzip to your favorite directory.
  2. Execute bin/windup, with the --input parameter set to your source directory or application archive.
  3. Make sure to set your --packages parameter to your Java package structure.
  4. Make sure to set the --source true flag.
  5. Make sure to set your --output parameter to your favorite output directory for the report.
  6. Execute windup
  7. View the report!
